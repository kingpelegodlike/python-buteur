# python-buteur
LE BUTEUR

version number: 0.1
author: Xavier Duval

## Overview

Electronic adaptation of the board game 'Le Buteur'.

## Installation / Usage

To install use pipenv python 3.10:

    $ pipenv sync

## Usage

    $ pipenv run python .\buteur.py [options]

## Unitary tests

    $ pipenv run pytest --cov-report html .\tests\
